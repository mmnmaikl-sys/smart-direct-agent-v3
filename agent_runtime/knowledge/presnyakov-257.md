# Presnyakov: 257 приёмов оптимизации Яндекс.Директа (Wave 1 subset)

> Полный список — Wave 2. Пока — ключевые приёмы для запуска и первичной оптимизации БФЛ-кампаний.

## Семантика и ключи

### Чистка мусорного трафика
- **Приём #1 — UTM-гигиена.** Каждая кампания имеет уникальный `utm_campaign`; группы — `utm_content`. Если UTM совпали у двух разных объявлений — attribution ломается, CPA считается криво.
- **Приём #2 — Минус-слова по кластеру запросов.** Раз в неделю выгружать «поисковые запросы» → группировать по Jaccard ≥0.5 → кластеры с CTR<0.5% или CR=0 целиком в минуса.
- **Приём #3 — Фразовое соответствие для ключей >3 слов.** Квадратные скобки `[банкротство физлиц Уфа]` на длинных ключах — Яндекс не добавит «мусор». Широкое — только на коротких.
- **Приём #4 — Регулярный аудит на "как сделать самостоятельно".** Для БФЛ это DIY-запросы (не наши клиенты). В минус сразу.

### Кластеризация
- **Приём #5 — Pillar → Hub → Spoke.** «Банкротство физлиц» (pillar) → «через МФЦ», «через арбитраж» (hub) → «сколько стоит», «сроки» (spoke). Одна группа = один интент.
- **Приём #6 — Региональные подкамп.** Башкортостан / Татарстан / Удмуртия = отдельные campaigns с RegionId. Не смешивать в одной — разные CPA / конверсии.

## Ставки и бюджет

### Автостратегии
- **Приём #7 — Не трогать автостратегию 48 часов после запуска.** Яндекс обучается. Любое изменение target CPA / budget ресетит обучение.
- **Приём #8 — `avg_cpa` ставим на 120-150% от желаемого.** Алгоритм «целится» ниже, не выше. Ставишь 1000 → получаешь 800-1200. Ставишь 800 → получаешь мало показов.
- **Приём #9 — Переход с «Максимум кликов» на «Оплата за конверсию» только после 10 конверсий/неделю.** Иначе Яндекс не найдёт закономерность.
- **Приём #10 — Недельный бюджет = 7 × дневной для ровности.** Автостратегия не любит резких смен лимита.

### Ручные ставки
- **Приём #11 — Stop-loss на кампанию.** Если spend_daily > 1.5× avg_7d — suspend + alert. Не ждать конца дня.
- **Приём #12 — P90 от истории ставок — потолок.** Новая ставка выше p90 за 30 дней = подозрение на баг в brain. Reject.

## Сезонность БФЛ

- **Приём #13 — Пик заявок на БФЛ: сентябрь-ноябрь, январь-март.** Летом спад на 30-40%. Закладывай в budget_plan.
- **Приём #14 — Квартальные отчёты ФССП (апрель, июль, октябрь) → всплеск запросов «арест счёта».** Заранее (за 2 недели) добавить объявления.
- **Приём #15 — Выплаты маткапитала, 13-й зарплаты (декабрь) — рост «закредитованности».** Готовь креативы про «новогодние долги».

## Объявления и креативы

- **Приём #16 — 3 объявления на группу минимум.** Яндекс выбирает лучший по CTR; меньше 3 = не с чем сравнивать.
- **Приём #17 — Первый заголовок до 33 симв, второй до 30.** Обрезается на мобильных иначе.
- **Приём #18 — Быстрые ссылки = 8 штук всегда.** Полный комплект CTR +15%.
- **Приём #19 — Уточнения (до 25 симв каждое) — 4 штуки.** Заполняют SERP.
- **Приём #20 — Не вставлять цены в заголовок без динамики.** Цена устарела → ФАС. Лучше «от» или «индивидуально».

## Измерения и отчётность

- **Приём #21 — Ежедневный grab в 10:00 МСК.** Вчерашний день уже стабилен. Не делать snapshot в 23:00 — конверсии ещё обсчитываются.
- **Приём #22 — Считать CPA по Bitrix (факт), не по Метрике (цель).** Метрика видит «конверсия цели», Bitrix — «деал». Расхождение 15-25% нормально.
- **Приём #23 — Lag на офлайн-конверсии 3-14 дней.** Не закрывать гипотезу "CPA=1500" через 3 дня — deals ещё поступают.

## Отладочные приёмы

- **Приём #24 — `query_analyzer` раз в 3 дня.** Не ежедневно — шум; не раз в месяц — успеваешь слить бюджет.
- **Приём #25 — Flip-flop detection: если за 14 дней вернулись к исходному состоянию — блок на 30 дней.** Агент «колеблется» = не уверен.
- **Приём #26 — Before/after snapshot перед любым SET.** Без baseline невозможно измерить impact.

## Региональная кастомизация (БФЛ-специфика)

- **Приём #27 — Башкортостан: «Уфа», «Стерлитамак», «Нефтекамск» — крупные города с отдельными кампаниями.** Малые посёлки — одной объединённой.
- **Приём #28 — Татарстан: Казань отдельно от остальной республики.** CPA в Казани на 30% ниже (выше конкуренция, но выше качество трафика).
- **Приём #29 — Удмуртия: низкая конкуренция, но и низкий объём.** РСЯ работает лучше Поиска для этого региона.

## Anti-patterns (что НЕ делать)

- **Приём #30 — Не копировать объявления из шаблонов конкурентов.** Яндекс видит одинаковые тексты → снижает рейтинг. Уникальный 2-й заголовок обязателен.
